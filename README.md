# MyDaily

## 计划内容：

1. 首页显示知乎日报文章
2. 查询天气，股票，快递，翻译等
3. 消费记录，分析消费，可设置每月上限
4. 番茄时钟
5. 备忘日程
5. 笔记本
6. 私密日记
